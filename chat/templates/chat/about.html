About This Project

Intelligent multi-domain support assistant powered by Google Gemini & Django.

Overview

This platform is a fully functional AI-powered support system capable of assisting users across multiple domains including:

  • Technical Support
  • Healthcare Information Assistance
  • Customer Support Queries
  • General Question Answering

Built using Django, MongoDB, and Gemini 2.0/2.5 models, the system provides fast, accurate and intelligent responses with persistent chat history, dashboard analytics, and a clean modern UI.

Key Features

✓ Multi-domain AI Support

Healthcare, Tech, Customer Support, and more.

✓ Real-time Gemini AI Responses

Powered by Google's most advanced Gemini models.

✓ Conversation History

All chats are saved to MongoDB automatically.

✓ Dashboard & Analytics

Track usage, activity, and system performance.

✓ Clean Modern UI

Responsive layout with sidebar, navbar & themes.

✓ Extensible Design

Add more pages, modes, or features easily.

Technologies Used

  • Django 6 – Backend framework
  • MongoDB – NoSQL database for chat history
  • Google Gemini API – AI model for generating responses
  • HTML, CSS, JS – Modern frontend with responsive layout
  • Fetch API – For AJAX calls to Django
