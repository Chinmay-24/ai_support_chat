Help & Documentation

How to use

  1. Open Chat to start or continue a conversation.
  2. Use the New button to start a fresh chat session.
  3. Export or download conversations using the topbar buttons.
  4. View stored conversations on the Conversations page.

Troubleshooting

  • If chat returns an error, check Django console for tracebacks.
  • Ensure MongoDB is running locally and `MONGODB_DB_NAME` is set in `settings.py`.
  • If API key is invalid, rotate it in Google AI Studio and update `settings.py`.
